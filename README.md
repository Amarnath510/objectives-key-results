# Objectives & Key Results

- Repository: https://github.com/Amarnath510/objectives-key-results/tree/master

### Steps to run:
- Download and cd to the repo
- npm i
- ng serve
- Open http://localhost:4200/


### Stack:
- Angular framework
- HTML semantics
- SCSS and Bootstrap
- BEM methodology
- Lodash utility
- RxJS
